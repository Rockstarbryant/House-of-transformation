# House of Transformation Church Website

A modern, full-stack church management website built with React and Node.js.

## Features

- User Authentication
- Live Streaming
- Sermon Library
- Blog/News
- Photo Gallery
- Kids Zone
- Volunteer Portal
- Event Management
- Admin Dashboard
- AI Chatbot
- Analytics

## Tech Stack

**Frontend:**
- React 18
- React Router v6
- Tailwind CSS
- Lucide Icons
- Axios

**Backend:**
- Node.js
- Express
- MongoDB
- JWT Authentication

## Installation

### Frontend
\`\`\`bash
npm install
npm start
\`\`\`

### Backend
\`\`\`bash
cd server
npm install
npm run dev
\`\`\`

## Environment Variables

Create `.env` files in root and server directories (see `.env.example`)

## Deployment

- Frontend: Vercel/Netlify
- Backend: Heroku/Railway
- Database: MongoDB Atlas
